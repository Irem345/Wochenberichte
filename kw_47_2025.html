Java Script

Java Script

Diese Woche haben wir intensiv mit JavaScript gearbeitet. JavaScript zählt zu den wichtigsten Programmiersprachen weltweit. Sie wird vor allem genutzt, um Webseiten interaktiv und lebendig zu gestalten. Während HTML das Gerüst einer Seite bildet und CSS das Aussehen bestimmt, sorgt JavaScript dafür, dass etwas passiert: Bewegungen, Animationen, Klickfunktionen oder der Austausch von Daten mit einer API werden beispielsweise durch JavaScript ermöglicht. Mit JavaScript können Spiele programmiert, Apps entwickelt, Formulare geprüft oder ganze Webseiten gesteuert werden. Die Sprache ist leicht zu erlernen und wird von allen modernen Browsern unterstützt. Deshalb ist JavaScript ideal für den Einstieg in die Welt der Programmierung.

Was ist Java Script?

JavaScript (JS) ist eine Skriptsprache, die hauptsächlich in Webbrowsern ausgeführt wird. Sie ermöglicht die Erstellung dynamischer und interaktiver Webseiten, indem sie Entwicklern die Möglichkeit gibt, auf Ereignisse zu reagieren, Benutzereingaben zu verarbeiten und den Inhalt einer Webseite während der Laufzeit zu ändern. Im Gegensatz zu statischem HTML und CSS verleiht JavaScript Webseiten Leben und ermöglicht ihnen gewisse Funktionalität.

Unterschied zwischen Java und JavaScript

Java und JavaScript sind zwei völlig verschiedene Programmiersprachen, auch wenn ihre Namen ähnlich klingen. Java ist eine objektorientierte, stark typisierte Sprache, die für grosse Softwareprojekte, Serveranwendungen, Desktopprogramme und Android-Apps genutzt wird. Java-Code muss vor der Ausführung kompiliert werden und läuft dann auf der Java Virtual Machine (JVM). JavaScript ist hingegen eine dynamische, flexibel typisierte Skriptsprache, die hauptsächlich im Browser ausgeführt wird und Webseiten interaktiv macht. Sie ermöglicht Animationen, Klickfunktionen, API-Verbindungen und vieles mehr, ohne dass eine vorherige Kompilierung erforderlich ist. Technisch, strukturell und im Einsatzgebiet haben Java und JavaScript somit fast nichts gemeinsam.

Fazit

Nach einer Woche JavaScript fühle ich mich sicherer. Ich habe gemerkt, wie wichtig JavaScript für Webseiten ist. Damit kann man schnell interaktive Elemente erstellen. Am Anfang war vieles neu und etwas verwirrend. Aber dann habe ich gemerkt, dass JavaScript eigentlich sehr logisch aufgebaut ist. Besonders spannend fand ich, dass man mit einfachen Befehlen Buttons, Animationen oder ganze Abläufe steuern kann. In der letzten Woche habe ich gelernt, dass man JavaScript für viele verschiedene Projekte benutzen kann.

Samichlaus und Schmutzli

Samichlaus

Der Samichlaus ist eine der bekanntesten und beliebtesten Figuren in der Schweizer Adventszeit. Viele denken dabei an den 6. Dezember, aber auch an warme Kindheitserinnerungen, Familienmomente und alte Traditionen. Was viele jedoch nicht wissen: Der Samichlaus hat eine lange Geschichte, die tief in europäischen Bräuchen verwurzelt ist. Seine Figur basiert auf dem heiligen Nikolaus, einem Bischof aus dem 4. Jahrhundert, der für seine Grosszügigkeit und Hilfsbereitschaft bekannt war. Aus dieser historischen Figur hat sich im Laufe der Jahrhunderte der "Samichlaus" entwickelt, wie wir ihn heute in der Schweiz kennen.

Der Samichlaus trägt typischerweise ein langes rotes oder braunes Gewand, einen weissen Bart und ein grosses Buch, in dem er aufschreibt, wie sich die Kinder während des Jahres verhalten haben. Oft ist auch der Schmutzli dabei, eine Figur, die früher für Strenge stand, heute aber meistens freundlich wirkt und einfach zum Ritual dazugehört. Gemeinsam besuchen sie Familien, Kindergärten oder Schulklassen. Statt grosser Geschenke bringt der Samichlaus vor allem kleine Leckereien wie Erdnüsse, Mandarinen, Schokolade und Lebkuchen. Dinge, die traditionell in einem Stoff oder Papiersäckli verteilt werden.

Ein weiterer spannender Teil dieser Tradition ist der Samichlaus-Vers. Viele Kinder lernen im Voraus einen kurzen Spruch auswendig, den sie dem Samichlaus aufsagen. Dafür gibt's Lob, ein Lächeln oder eben ein gefülltes Säckli. Der Nikolausbesuch ist also nicht nur ein schöner Moment, sondern auch ein kleines Ritual voller Mut, Freude und ein bisschen Nervosität.

Verbindung zur Türkei: Was viele nicht wissen: Der heilige Nikolaus kommt ursprünglich aus der heutigen Türkei. Nikolaus wurde nämlich in Patara geboren, einer antiken Stadt an der türkischen Südküste. Später hat er als Bischof von Myra gearbeitet, das heute Demre heisst. Die Geschichte des Schweizer Samichlaus geht also tatsächlich auf die Türkei zurück. Die Tradition kommt ursprünglich aus einer Region, die heute ein wichtiger Teil der türkischen Geschichte ist. Sie hat sich dann über viele Länder verbreitet und wurde immer wieder angepasst.
