HTML

HTML steht für Hypertext Markup Language. Auf Deutsch heisst das Hypertext-Auszeichnungssprache. HTML ist keine Programmiersprache, sondern eine Sprache, mit der man die Struktur einer Webseite beschreibt. Also sowie die Basis für eine Webseite. Viele Menschen denken, HTML eine Programmiersprache ist, aber das stimmt nicht. HTML ist dafür da, Texte, Bilder, Links und andere Elemente auf einer Webseite anzuzeigen und zu ordnen.HTML ist sehr wichtig für das World Wide Web. Ohne HTML gäbe es keine Webseiten wie YouTube, Wikipedia oder Online-Shops. Jede einzelne Seite im Internet ist ein HTML-Dokument. Diese Seiten sind oft miteinander verlinkt, also durch sogenannte Hyperlinks verbunden. So kann man von einer Seite zur nächsten springen. Ein Browser wie Chrome oder Firefox liest den HTML-Code und zeigt die Inhalte für den Benutzer an. HTML ist die Grundlage für jede Webseite. Wenn man eine Webseite bauen möchte, beginnt man mit HTML. Später kann man andere Sprachen wie CSS (für Design) oder JavaScript (für Funktionen) hinzufügen. Aber ohne HTML funktioniert keine Webseite. Deshalb ist HTML ein sehr wichtiger Teil der Digitalisierung im 20. und 21. Jahrhundert. Es ist einfach zu lernen und ein guter Start für alle, die eine Webseite Programmieren möchten.

CSS

CSS bedeutet „Cascading Style Sheets“. Auf Deutsch nennt man das „kaskadierende Stilvorlagen“. CSS ist eine Sprache für das Design von Webseiten. Mit CSS kann man das Aussehen und das Layout ändern. HTML baut die Struktur der Seite, CSS macht sie schön.

Mit CSS kann man viele Dinge machen:

  • Farben von Text und Hintergrund ändern
  • Schriftarten und Schriftgrössen auswählen
  • Abstände und Positionen von Elementen bestimmen

CSS hilft auch, Webseiten für verschiedene Geräte anzupassen, zum Beispiel für Handy oder Computer. Man kann CSS direkt im HTML schreiben oder in einer extra Datei speichern. Diese Datei nennt man externes Stylesheet.Ohne CSS wären Webseiten langweilig und schwer zu lesen. Mit CSS sehen sie modern und freundlich aus. Auch Anfänger können mit CSS arbeiten und schnell gute Ergebnisse sehen. CSS ist sehr wichtig für Webdesign und wird von vielen Webentwicklern benutzt.

Fazit

Ich habe gelernt, wie man beides benutzt. Es war spannend zu sehen, wie Webseiten aufgebaut sind und wie man sie gestalten kann.
